# URL Shortener on Go + Svelte

URL shortener built with Go and Svelte.

Project was built as a practice project to learn Go and Svelte.

## Main Features

- Shorten URLs 
- Check number of redirects via short URL code

![Application Preview](assets/preview.png)

## Stack

- Go
- Svelte
- SQLite
- Vite
- TS

## Running Locally

To run this project locally, clone the repository and run the following commands:

```bash
# Backend
cd backend
go run .
```

```bash
# Frontend
cd frontend
npm install
npm run dev
```
